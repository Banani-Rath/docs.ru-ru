# Обработка XML в Visual Basic
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

*XML\-литералы* можно использовать для загрузки XML из внешнего источника, таких как строка, файл или поток.  Затем можно использовать [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] для управления XML и [!INCLUDE[vbteclinqext](../../../../csharp/getting-started/includes/vbteclinqext_md.md)] для составления запросов XML.
