# Операторы Select недопустимы в окне интерпретации
Операторы `Select` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30718  
  
### Исправление ошибки  
  
-   Удалите операторы `Select` из отлаживаемого кода.
