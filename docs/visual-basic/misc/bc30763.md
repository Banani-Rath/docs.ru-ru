# Целевая версия .NET Compact Framework не поддерживает использование модификатора Ansi, Auto или Unicode
Версия .NET Compact Framework, с которой вы работаете, не поддерживает модификаторы `Ansi`, `Auto` и `Unicode`.  
  
 **Идентификатор ошибки:** BC30763  
  
### Исправление ошибки  
  
-   Удалите этот модификатор.
