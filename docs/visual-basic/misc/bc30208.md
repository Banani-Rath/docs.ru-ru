# Требуется Compare
В операторе `Option` указывается `Text` или `Binary`, но не `Compare`.  
  
 **Идентификатор ошибки:** BC30208  
  
### Исправление ошибки  
  
-   Добавьте ключевое слово `Compare` в оператор `Option`, например `Option Compare Binary`.
