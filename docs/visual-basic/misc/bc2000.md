# Ошибка инициализации
Произошла ошибка при инициализации программы. Это может быть результатом отмены установки в процессе ее выполнения.  
  
 **Идентификатор ошибки:** BC2000  
  
### Исправление ошибки  
  
-   Повторите попытку установки.
