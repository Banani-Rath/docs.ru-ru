# При вызове метода не возвращено значение
Вызываемый метод не возвращает значение.  
  
 **Идентификатор ошибки:** BC30723  
  
### Исправление ошибки  
  
-   Убедитесь, что вызываемый метод является функцией, которая возвращает значение.
