# Комментарий XML должен быть первым оператором в строке
Комментарий XML должен быть первым оператором в строке. Комментарий XML будет пропущен.  
  
 Комментарий XML помещен после другого элемента языка, что не допускается.  
  
 **Идентификатор ошибки:** BC42302  
  
### Исправление ошибки  
  
-   Переместите комментарий XML на новую строку.
