# Операторы Resume недопустимы в окне интерпретации
Операторы Resume можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30714  
  
### Исправление ошибки  
  
-   Удалите операторы `Resume` из отлаживаемого кода.
