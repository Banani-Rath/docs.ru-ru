# Требуется целочисленная константа.
Указана директива `#ExternalSource`, в которой второй аргумент не является целочисленным литералом. В этом контексте допустим только целочисленный литерал. Именованная константа или член перечисления не является допустимым.  
  
 **Идентификатор ошибки:** BC30204  
  
### Исправление ошибки  
  
1.  Используйте именованную константу или член перечисления вместо литерала.  
  
2.  Укажите целочисленный литерал в качестве второго аргумента директивы `#ExternalSource`.
