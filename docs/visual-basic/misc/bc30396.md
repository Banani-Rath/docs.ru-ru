# &quot;&lt;спецификатор&gt;&quot; недопустим в объявлении перечисления
Использовался спецификатор, недопустимый в объявлении `Enum`.  
  
 **Идентификатор ошибки:** BC30396  
  
### Исправление ошибки  
  
1.  Удалите спецификатор.
