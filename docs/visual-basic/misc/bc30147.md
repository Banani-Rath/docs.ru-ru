# Операторы Return недопустимы в окне интерпретации
Оператор `Return` выполняет ветвление и не разрешен в контексте отладки.  
  
 **Идентификатор ошибки:** BC30147  
  
### Исправление ошибки  
  
-   Не используйте оператор `Return` в окне **интерпретации**.
