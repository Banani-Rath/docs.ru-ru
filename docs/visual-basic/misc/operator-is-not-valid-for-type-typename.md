# Недопустимый оператор для типа &quot;&lt;имя_типа&gt;&quot;
Оператор недопустим для типов.  
  
### Исправление ошибки  
  
-   Убедитесь в том, что оператор является допустимым для типов, с которыми совершаются операции.
