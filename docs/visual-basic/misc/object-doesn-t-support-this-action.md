# Объект не поддерживает это действие (Visual Basic)
Ссылка на метод или свойство, которое не поддерживается данным объектом.  
  
### Исправление ошибки  
  
1.  Дополнительные сведения см. в документации по объекту.  
  
2.  Убедитесь, что имена свойств и методов написаны правильно.
