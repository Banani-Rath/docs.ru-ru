# Общий конструктор Sub New не может иметь параметров
Вы включили параметр или параметры в общий оператор `Sub New`, что недопустимо.  
  
 **Идентификатор ошибки:** BC30479  
  
### Исправление ошибки  
  
-   Удалите параметр или параметры.
