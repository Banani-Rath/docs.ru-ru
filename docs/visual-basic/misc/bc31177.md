# Пробел не может быть указан здесь
Пробел или пробелы включены в имя литерала элемента XML.  
  
 **Идентификатор ошибки:** BC31177  
  
### Исправление ошибки  
  
-   Удалите пробел или пробелы из имени литерала элемента XML.
