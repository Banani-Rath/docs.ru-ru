# Ожидался символ &quot;-&quot;
Неверный синтаксис в этой строке.  
  
 **Идентификатор ошибки:** BC30601  
  
### Исправление ошибки  
  
-   Проверьте синтаксис кода, в котором произошла эта ошибка.
