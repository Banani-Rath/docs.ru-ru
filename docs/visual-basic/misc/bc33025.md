# Операторы преобразования не могут приводить к типу интерфейса
Оператор преобразования объявлен с типом интерфейса в качестве типа возвращаемых данных.  
  
 Во время компиляции [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] определяет возможность выполнения предопределенного преобразования любого ссылочного типа в любой интерфейс. Такое преобразование может вызвать сбой во время выполнения, однако компилятор не может предсказать результаты выполнения, поэтому он позволяет компилировать такие преобразования.  
  
 Так как компилятор считает, что это преобразование уже определено, он не позволяет переопределить его.  
  
 **Идентификатор ошибки:** BC33025  
  
### Исправление ошибки  
  
-   Полностью удалите это определение оператора. Он является предварительно определенным.
