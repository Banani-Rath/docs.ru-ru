# Диск не готов
Для указанного дисковода либо отсутствует диск, либо задвижка дисковода открыта.  
  
### Исправление ошибки  
  
1.  Проверьте указанный дисковод и вставьте диск при необходимости.  
  
2.  Закройте задвижку дисковода.
