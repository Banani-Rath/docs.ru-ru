# ByVal и ByRef нельзя использовать вместе
Ключевые слова `ByVal` и `ByRef` являются взаимоисключающими.  
  
 **Идентификатор ошибки:** BC30641  
  
### Исправление ошибки  
  
-   Укажите только один модификатор: `ByVal` или `ByRef`.
