# Эта система не предусматривает поддержку стандартов упрощенного китайского языка
Вы пытаетесь использовать член перечисления `VbStrConv.SimplifiedChinese`, но ваша система не поддерживает упрощенный китайский язык.  
  
### Исправление ошибки  
  
1.  Обновите систему для поддержки упрощенного китайского языка или запустите приложение на компьютере, который поддерживает этот языковой стандарт.
