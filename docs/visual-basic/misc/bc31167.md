# Требуется &quot;/&quot; для конечного тега XML
В конечном теге для XML\-литерала отсутствует необходимый символ "\/".  
  
 **Идентификатор ошибки:** BC31167  
  
### Исправление ошибки  
  
-   Добавьте символ "\/" в начало конечного XML\-тега для XML\-литерала.
