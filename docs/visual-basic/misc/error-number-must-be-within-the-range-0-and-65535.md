# Номер ошибки должен находиться в диапазоне от 0 до 65535
Номер ошибки выходит за пределы разрешенного диапазона от 0 до 65535.  
  
### Исправление ошибки  
  
-   Убедитесь, что номер попадает в допустимый диапазон.
