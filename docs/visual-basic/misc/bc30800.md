# Аргументы метода должны быть заключены в круглые скобки
Правила, управляющие вызовами процедур, стали проще в более новых версиях [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]. Все аргументы должны быть заключены в круглые скобки.  
  
 **Идентификатор ошибки:** BC30800  
  
### Исправление ошибки  
  
-   Заключите список аргументов в скобки; например:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
