# Значение BaudRate должно быть больше нуля
Аргумент `BaudRate`, указанный для метода `My.Computer.Ports.OpenSerialPort`, должен быть больше нуля.  
  
### Исправление ошибки  
  
-   Измените значение аргумента `BaudRate` на положительное число.
