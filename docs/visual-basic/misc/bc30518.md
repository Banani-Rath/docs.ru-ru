# Сбой при разрешении перегрузки, так как ни один из доступных &quot;&lt;метод&gt;&quot; не может быть вызван с данными аргументами: &lt;ошибка&gt;
Вы выполнили вызов перегруженного метода, но компилятор не может найти метод с соответствующим списком аргументов.  
  
 **Идентификатор ошибки:** BC30518  
  
### Исправление ошибки  
  
-   Проверьте указанный список аргументов.
