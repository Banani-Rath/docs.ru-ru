# &quot;&lt;спецификатор&gt;&quot; является недопустимым объявлением константы
Оператор `Const` содержит недопустимое ключевое слово. Оператор `Const` может содержать только ключевые слова `Friend`, `Private`, `Protected`, `Public` и `Shadows`.  
  
 **Идентификатор ошибки:** BC30233  
  
### Исправление ошибки  
  
1.  Убедитесь в том, что ключевое слово написано правильно.  
  
2.  Удалите недопустимое ключевое слово из оператора `Const`.
