# Операторы Throw недопустимы в окне интерпретации
Операторы `Throw` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30725  
  
### Исправление ошибки  
  
-   Удалите операторы `Throw` из отлаживаемого кода.
