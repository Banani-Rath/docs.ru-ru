# Значение DataBits должно быть больше 0
Аргумент `DataBits`, указанный для метода `My.Computer.Ports.OpenSerialPort`, должен быть больше нуля.  
  
### Исправление ошибки  
  
-   Измените значение аргумента `DataBits` на положительное число.
