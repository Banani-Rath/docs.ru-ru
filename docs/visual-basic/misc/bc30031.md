# Полное имя типа в пространстве имен не может быть длиннее &lt;число&gt; символов
Полное имя типа не может быть длиннее, чем указанное количество символов.  
  
 **Идентификатор ошибки:** BC30031  
  
### Исправление ошибки  
  
-   Укоротите имя типа.
