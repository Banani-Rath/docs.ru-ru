# Операторы объявления методов должны указываться первыми в логической строке
Оператор `Function` или `Sub` следует за разделителем операторов в виде двоеточия \(:\).`Function` или `Sub` должен быть единственным оператором в строке исходного кода.  
  
 **Идентификатор ошибки:** BC32009  
  
### Исправление ошибки  
  
-   Разместите несколько операторов в отдельных строках.
