# &quot;End AddHandler&quot; должен быть первым оператором в строке
Оператор `End AddHandler` следует за разделителем операторов в виде двоеточия \(:\).`End AddHandler` должен быть единственным оператором в строке исходного кода.  
  
 **Идентификатор ошибки:** BC31118  
  
### Исправление ошибки  
  
-   Разместите несколько операторов в отдельных строках.
