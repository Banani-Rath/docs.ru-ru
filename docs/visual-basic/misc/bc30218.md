# Требуется &quot;Lib&quot;
В операторе `Declare` имя библиотеки указано без ключевого слова `Lib`.  
  
 **Идентификатор ошибки:** BC30218  
  
### Исправление ошибки  
  
-   Добавьте ключевое слово `Lib` перед именем библиотеки.
