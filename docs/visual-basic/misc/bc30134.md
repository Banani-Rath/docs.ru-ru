# Метки в окне интерпретации являются недопустимыми.
Метки строки включают ответвления и не разрешены в контексте отладки.  
  
 **Идентификатор ошибки:** BC30134  
  
### Исправление ошибки  
  
-   Не следует вводить метку строки в окне **интерпретации**.
