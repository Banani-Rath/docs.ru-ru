# Операторы EndIf больше не поддерживаются; используйте вместо них End If
Оператор `EndIf` больше не поддерживается.  
  
 **Идентификатор ошибки:** BC30826  
  
### Исправление ошибки  
  
-   Используйте `End If` для завершения операторов `If`.
