# Объект My.Computer.Ports
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Предоставляет свойства и методы для доступа к последовательным портам компьютера.  
  
## Заметки  
 Дополнительные сведения о доступных методах и свойствах объекта `My.Computer.Ports` см. в разделе <xref:Microsoft.VisualBasic.Devices.Ports>.  
  
 Дополнительные сведения см. в разделе [Доступ к портам компьютера](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
>  Также можно использовать методы и свойства класса <xref:System.IO.Ports.SerialPort?displayProperty=fullName> для доступа к последовательным портам компьютера.
