# Ошибка компилятора CS0148
Делегат "делегат" не имеет допустимого конструктора  
  
 Вы импортировали и использовали управляемую программу \(использующую среду CLR .NET Framework\), созданную с помощью другого компилятора. Этот компилятор разрешил неправильно сформированный конструктор [делегата](../../csharp/language-reference/keywords/delegate.md). Дополнительные сведения см. в разделе [Делегаты](../../csharp/programming-guide/delegates/index.md).
