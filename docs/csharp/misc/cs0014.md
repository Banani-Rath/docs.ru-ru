# Ошибка компилятора CS0014
Не удалось найти требуемый файл "файл"  
  
 В системе отсутствует файл, необходимый для компилятора. Проверьте правильность пути. Если требуется системный файл [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)], то, возможно, необходимо исправить или полностью переустановить [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].
