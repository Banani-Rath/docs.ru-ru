# Ошибка компилятора CS0534
"функция1" не реализует наследуемый абстрактный член "функция2"  
  
 Если класс не является абстрактным, он должен реализовывать все [абстрактные](../../csharp/language-reference/keywords/abstract.md) члены базового класса.  
  
 Следующий пример приводит к возникновению ошибки CS0534:  
  
```  
// CS0534.cs namespace x { abstract public class clx { public abstract void f(); } public class cly : clx   // CS0534, no override for clx::f { // uncomment the following sample override to resolve CS0534 // override public void f() // { // } public static int Main() { return 0; } } }  
```
