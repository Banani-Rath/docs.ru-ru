# Ошибка компилятора CS1719
Ошибка при чтении файла ресурсов Win32 "имя файла" — "причина"  
  
 Попытка прочитать файл ресурсов Win32 окончилась неудачей по причине, указанной в сообщении об ошибке. Обычно в качестве причины указывается "файл не найден" или "доступ запрещен". Для устранения этой ошибки необходимо разрешить проблему, описанную в причине.
