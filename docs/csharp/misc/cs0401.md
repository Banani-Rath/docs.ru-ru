# Ошибка компилятора CS0401
Ограничение new\(\) должно быть последним указанным ограничением.  
  
 При использовании нескольких ограничений перечислите все прочие ограничения до ограничения new\(\).  
  
## Пример  
 В следующем примере возникает ошибка CS0401.  
  
```  
// CS0401.cs // compile with: /target:library using System; class C<T> where T : new(), IDisposable {}  // CS0401 class D<T> where T : IDisposable { static void F<U>() where U : new(), IDisposable{}   // CS0401 }  
```
