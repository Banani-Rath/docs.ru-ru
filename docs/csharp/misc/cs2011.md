# Ошибка компилятора CS2011
Не удается открыть файл ответов "файл"  
  
 При компиляции был указан файл ответов, однако компилятору не удалось обнаружить и открыть этот файл.  
  
 Дополнительные сведения о файлах ответов см. в разделе [@ \(Указание файла ответов\)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
