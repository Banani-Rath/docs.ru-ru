# Ошибка компилятора CS1565
Заданы несовместимые параметры: файл ресурсов Win32; значок Win32.  
  
 Не допускается указывать оба параметра компилятора [\/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) и [\/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) в одной компиляции.
