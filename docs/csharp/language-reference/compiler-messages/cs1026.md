# Compiler Error CS1026
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Требуется "\)"  
  
 Обнаружен незавершенный оператор.  
  
 Наиболее распространенная причина возникновения этой ошибки состоит в том, что во встроенное выражение на странице ASP.NET помещено не выражение, а оператор.  Например, следующий код является неверным:  
  
```  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 Правильный пример кода:  
  
```  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 Он интерпретируется следующим образом:  
  
```  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 Следующий пример приводит к возникновению ошибки CS1026:  
  
```  
// CS1026.cs  
#if (a == b   // CS1026, add closing )  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
